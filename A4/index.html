Temperature/Humidity

Step 1

Start by creating an EC2 server on AWS by pressing the EC2 buttonon the home page. Then go to Instances (running) page and select the "Launch Instance" button on the right.

Step 2:

From here a page will open with serval different settings. Go to the OS and change it to Unbuntu.

Choose Unbuntu Server 20.04 as the Amazon Machine Image.

Change the key pair to "Proceed without a key pair"

Change the configuration on the bottom of the page to 1x28

Finally press the Launch Instance at bottom of the page.

Step 3:

Now select your newly created instance. Scroll down and select the security tab. Select launch wizard and go into the inbound rules.

Press "edit Inbound Rules" by pressing "add rule" to change the port range on both.

Add "8080" and change the source to "Anywhere IPV4". Then add "1234" and chnage source "Anywhere IPV4."

Press save Rules to apply to Instance & return to the instnace page.

Step 4:

Go to your Instance summary and press connect. Enter the following commands to the HYYPServer.

Curl Bash command the class repo link

Npm install

npm install latest

sudo npm install -g n

sudo npm install -g http-server

Now you will clone the GitHub Repo via "git clone" plus the class link.

Step 5:

Now that you have cloned your GitHub, use ls & cd commands to navigate the server.js file.

Once there type node server.js to start running the server.

To view the server write the public IP with 8080/ at the end.

Step 6:

To make the TTGo Watch work, open the file DHT11Default in Arduino.

Check that he port is TTGo Watch and the watchUSB is connected with the temp sensor.

Make sure theblue light is on.

Upload any other needed files on to the watch & change the wifi & password to hotspot info.

Now change the URL to your Public IP address.

Make sure to change your pin to 25.

Finally upload the code & said temp should display on the web page.
